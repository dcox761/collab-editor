# Nested Document Example

This document lives inside a subfolder to demonstrate the **file tree navigation**.

## Why Nested Folders?

Organising documents in folders helps keep your workspace tidy:

- Group related documents together
- Create hierarchies for complex projects
- Keep your sidebar manageable

## Example Structure

```
docs/
├── welcome.md
└── example/
    └── nested-doc.md
```
