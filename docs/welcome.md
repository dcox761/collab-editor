# Welcome to Collab Editor

This is a **collaborative Markdown editor** built with BlockNote.

## Features

- 📂 **File Browser** — Browse and manage your documents
- ✏️ **Block Editor** — Rich Markdown editing with BlockNote
- 🗂️ **Multi-Tab Support** — Open multiple documents at once
- 💾 **Save with Ctrl+S** — Your changes persist to disk

## Getting Started

1. Click a file in the sidebar to open it
2. Edit using the block-based editor
3. Press **Ctrl+S** to save your changes

## Markdown Support

You can write:

- **Bold text**
- *Italic text*
- `Code snippets`
- [Links](https://example.com)

> Blockquotes work too!
